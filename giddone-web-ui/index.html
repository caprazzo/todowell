Write Code on Github
See TODOs on Giddone

Get Started:

1. Add hook to your repo

  • Browse to any of your github repos
  • Browse to the settings page
  • Click on "Service Hooks"
  • Click on "WebHook URLs"
  • Enter http://gitdone-receive-hook.herokuapp.com/github/post-receive-hooks
  • Click on "Test Hook"

2. Done

To see your TODOs browse to https://s3.amazonaws.com/giddone/USER/REPO/master

For example: https://s3.amazonaws.com/giddone/mcaprari/todowell/master

We'll update your todos page every time you push to this repo
